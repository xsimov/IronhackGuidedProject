Creating the Models

In our application, we will need to create four models (remember: User, Bag, Item, Trip). It would be great here read the Naming Conventions.

For creating a model, we need to use the following command:

$ rails g model user

The database has to be configured for the classes we are going to create. Everytime we create a model, a file called "migration" will be created in the folder "db/migrate" with the following format:

"db/migrate/XXXXXXXXXXXXXX_create_yourmodel.rb"

There is where we have to edit the rows our database table is going to have. But before modifying anything, let's create all the models we need.

$ rails g model bag

$ rails g model item

$ rails g model trip

Now, we have to go to the migrations of each migration to edit the table rows:

#Users
class CreateUsers < ActiveRecord::Migration
  def change
    create_table :users do |t|
      t.string :username
      t.string :password
      t.string :email
      t.timestamps
    end
  end
end

#Bags
class CreateBags < ActiveRecord::Migration
  def change
    create_table :bags do |t|
      t.string :bagname
      t.text :description
      t.timestamps
    end
  end
end

#Items
class CreateItems < ActiveRecord::Migration
  def change
    create_table :items do |t|
      t.string :name
      t.string :icon
      t.string :category
      t.timestamps
    end
  end
end

#Trips
class CreateTrips < ActiveRecord::Migration
  def change
    create_table :trips do |t|
      t.string :city
      t.string :country
      t.datetime :from_date
      t.datetime :to_date
      t.text :description
      t.timestamps
    end
  end
end

Cool! But there is something missing... the relations between the tables. If you go to the chapter "Design the database", you will se that there are some other fields, like for example, a user has a lot of bags, but a bag only has one trip... let's declare that.

This is what is called "foreign keys". It is a relation between different tables in a database. Now, we need to go to the models and add the relationship. I did it like that:

class User < ActiveRecord::Base
    has_many: bags
end

class Bag < ActiveRecord::Base
    belongs_to :location
    has_many :items
    has_one :trip
end

class Item < ActiveRecord::Base
    belongs_to :bag
end

class Trip < ActiveRecord::Base
    belongs_to :bag
end

Now we have the migrations created, let's migrate it. Type the following command in the terminal:

$ rake db:migrate
